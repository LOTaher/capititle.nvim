# `capititle.nvim`

A Neovim plugin to easily capitalize titles with a single key mapping.

## Documentation

Coming soon...

## Usage

```lua
-- lazy.nvim
return {
	"lotaher/capititle.nvim",
	config = function()
		vim.keymap.set("v", "<leader>ct", vim.cmd.Capititle)
	end,
}
```
